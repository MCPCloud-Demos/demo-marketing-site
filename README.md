# demo-marketing-site

> **Throwaway demo repo.** This repository exists as **seed data** for the
> [MCPCloud](https://mcpcloud.sh) GitHub showcase server — a real, installable
> MCP server that wraps the GitHub REST API and lets agents work on demo
> issues and PRs without touching anyone's real codebase.
>
> Everything in this repo (issues, PRs, comments, branches) is **reset
> nightly**. Do not link to or rely on any content here. If you found this
> repo organically, head to <https://mcpcloud.sh> — we'd love to show you the
> real product.

## What lives here

A pretend marketing site for a fictional company. The seed includes a mix of:

- Open bugs (broken link, copy errors, image regressions)
- Stale review requests (no reviewer assigned, > 2 days old)
- Closed-as-done issues so historical queries return non-empty results

This shape mirrors the [Forge look-alike](https://github.com/MCPCloud-Demos/forge)
so the real-GitHub showcase narrative is identical to the look-alike walkthrough.
